## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Download MobileView Extension on VSCode
3.  Start the app

   ```bash
    npx expo start
   ```
4. Copy in the web link into the MobileView (click MobileView, click "Iphone X", input in the search bar `https://localHost....` <- something like that)
5. VIEW!!!!
